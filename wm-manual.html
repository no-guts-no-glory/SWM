Window manager shortcuts


    Ctr+F1           web-browser
    Ctr+F2           alternate web-browser
    Ctr+F3           take a screenshot

    WIN+Ctr+q        quit     window manager
    WIN+Ctr+k        kill     current window
    WIN+Ctr+ALT+r    reboot   system
    WIN+Ctr+ALT+u    shutdown system

    WIN+return       start a new terminal
    WIN+space        start quick program launcher

    WIN+period       activate next window
    WIN+tab          activate next window
    WIN+comma        activate previous window
    WIN+SHIFT+tab    activate previous window

    WIN+1            goto 1st  window
    WIN+2            goto 2nd  window
    WIN+3            goto 3rd  window
    WIN+4            goto 4th  window
    WIN+5            goto 5th  window
    WIN+6            goto 6th  window
    WIN+7            goto 7th  window
    WIN+8            goto 8th  window
    WIN+9            goto 9th  window
    WIN+0            goto 10th window

    WIN+k            cascade  windows
    WIN+m            maximize window
    WIN+n            minimize window
    WIN+v            maximize window vertically
    WIN+h            maximize window horizontally
    WIN+[            snap window to West
    WIN+]            snap window to East
    WIN+t            snap window to North
    WIN+b            snap window to South

    WIN+q            move to North-west
    WIN+w            move to North
    WIN+e            move to North-east
    WIN+a            move to West
    WIN+s            move to Centre
    WIN+d            move to East
    WIN+z            move to South-west
    WIN+x            move to South
    WIN+c            move to South-east

    WIN+SHIFT+q      move and resize to North-west
    WIN+SHIFT+w      move and resize to North
    WIN+SHIFT+e      move and resize to North-east
    WIN+SHIFT+a      move and resize to West
    WIN+SHIFT+s      move and resize to Center
    WIN+SHIFT+d      move and resize to East
    WIN+SHIFT+z      move and resize to South-west
    WIN+SHIFT+x      move and resize to South
    WIN+SHIFT+c      move and resize to South-east

    WIN+Up           shift up
    WIN+Down         shift down
    WIN+Left         shift left
    WIN+Right        shift rigth

    WIN+SHIFT+Up     resize upward
    WIN+SHIFT+Down   resize downward
    WIN+SHIFT+Left   resize left
    WIN+SHIFT+Right  resize rigth

    WIN+SHIFT+period goto next desktop
    WIN+SHIFT+comma  goto previous desktop

    F1               goto desktop 1
    F2               goto desktop 2
    F3               goto desktop 3
    F4               goto desktop 4
